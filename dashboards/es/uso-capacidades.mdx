## Imagen

### Usando Markdown

La [sintaxis de markdown](https://www.markdownguide.org/basic-syntax/#images) te permite agregar imágenes usando el siguiente código

```md
![título](/ruta/imagen.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser menor a 5MB. De lo contrario, recomendamos alojarlo en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Luego puedes usar esa URL e incrustarla.

### Usando Incrustaciones

Para obtener más personalización con imágenes, también puedes usar [incrustaciones](/writing-content/embed) para agregar imágenes

```html
<img height="200" src="/ruta/imagen.jpg" />
```

### iFrames

Carga otra página HTML dentro del documento. Se usa más comúnmente para incrustar videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
