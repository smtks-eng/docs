## Básico

### Código en Línea

Para denotar una `palabra` o `frase` como código, enciérrala en comillas invertidas (\`).

```
Para denotar una `palabra` o `frase` como código, enciérrala en comillas invertidas (`).
```

### Bloque de Código

Usa [bloques de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código en tres comillas invertidas y sigue las comillas iniciales con el lenguaje de programación de tu fragmento para obtener resaltado de sintaxis. Opcionalmente, también puedes escribir el nombre de tu código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
