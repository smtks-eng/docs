## Setting up with Semantiks

Open [Semantiks-Console](https://consola.semantiks.com/login) log-in and [create](https://consola.semantiks.com/signup) an account

![Screenshot 2025-06-13 at 3.33.05 p.m..png](/images/Screenshot2025-06-13at3.33.05p.m..png)

After filling the necessary information you are going to recibe the corrsponding notification via email

![Screenshot 2025-06-13 at 3.35.16 p.m..png](/images/Screenshot2025-06-13at3.35.16p.m..png)

The mail should contain a message like this one with a link to continue the account creation

![Screenshot 2025-06-13 at 3.41.52 p.m..png](/images/Screenshot2025-06-13at3.41.52p.m..png)

### CREATING YOUR ACCOUNT

First fill the member information.

![Screenshot 2025-06-13 at 3.47.56 p.m..png](/images/Screenshot2025-06-13at3.47.56p.m..png)

### ORGANIZATION SETTING

Fill the Organization information.

<Note>
  If you already have an account created and associated to an organization please go directly to the ORGANIZATIONS section of the doc where you can find how to add users to an existing organization
</Note>

![Screenshot 2025-06-13 at 4.28.21 p.m..png](/images/Screenshot2025-06-13at4.28.21p.m..png)

### AGENT FIRST SET-UP

After filling the **organization information** select an agent type

<Note>
  If you want to know more about the **agent types** please see the corresponding data in the Agents section
</Note>

![Screenshot 2025-06-14 at 3.10.41 p.m..png](/images/Screenshot2025-06-14at3.10.41p.m..png)

Set the basic configuration for your agent

![Screenshot 2025-06-14 at 11.25.28 p.m..png](/images/Screenshot2025-06-14at11.25.28p.m..png)

Add some knowledge to the agent

![Screenshot 2025-06-14 at 11.28.05 p.m..png](/images/Screenshot2025-06-14at11.28.05p.m..png)

### NEXT-STEPS

You are all set and ready to go. You might be added to a \*\*Wait-List \*\*don't worry the time of response would be quick
