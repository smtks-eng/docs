title: "Performance"
description: "Track your live-chat team's throughput, response times, and individual agent performance in one consolidated dashboard."
---

The **Performance** dashboard gives contact-center managers a real-time view of how the human support team is handling escalated conversations. Use it to spot bottlenecks, compare agent workloads, and drill into any specialist's conversation history.

## Filters

All charts and tables on the page respond to a shared filter bar at the top-right:

| Filter | Description |
|---|---|
| **Granularity** | Choose how time buckets are grouped: **Day**, **Week**, or **Month**. |
| **Date range picker** | Pick a custom start and end date for the analysis window. |
| **Date presets** | Quick-select buttons for **1 d**, **3 d**, **7 d**, **14 d**, or **30 d**. Selecting a preset clears any custom range. |
| **Agent** | Filter the throughput chart to a single specialist, or choose **All specialists** to see the full stacked view. |

<Tip>
Combine a custom date range with the **Month** granularity to review an entire quarter at a glance.
</Tip>

---

## Throughput chart

The main chart occupies the left two-thirds of the page and shows **conversations handled per time bucket**.

### All specialists view

When the agent filter is set to **All specialists**, the chart renders a **stacked bar chart** where each colored segment represents a different agent. The agent with the highest volume sits at the bottom of the stack; smaller contributors stack on top.

- A **color-coded legend** appears below the chart, mapping each agent's name to their bar color.
- **Hover** over any bar to see a custom tooltip listing every agent who handled conversations that day, sorted by volume (largest first), plus a **Total handled** line at the bottom.

The first seven agents in the leaderboard receive fixed brand colors so they are easy to recognize across both the chart and the ranking table. Agents beyond the top seven receive a stable, automatically generated hue based on their ID.

### Single agent view

When a specific agent is selected, the chart shows only that agent's daily counts as simple bars, colored to match their leaderboard rank.

## Leaderboard

The right third of the page shows an **agent leaderboard** table, ranked by the number of conversations handled in the selected period. Each row displays:

| Column | Description |
|---|---|
| **#** | Rank position |
| **Agent** | The specialist's name |
| **Handled** | Total conversations handled |
| **Avg First Response** | Average time to the agent's first reply |
| **Avg Resolution** | Average time to resolve (close) a conversation |

Click any row to open a **side panel** with that agent's individual conversations for the selected period, so you can read transcripts and review outcomes without leaving the dashboard.

<Tip>
Use the leaderboard to quickly identify top performers and agents who may need coaching — look for outliers in response or resolution times.
</Tip>

## Common workflows

- **Daily stand-up** — Set the date preset to **1 d** and granularity to **Day** to review yesterday's workload distribution.
- **Weekly review** — Use the **7 d** preset to compare agent throughput over the past week.
- **Deep dive on a specialist** — Select an agent from the filter dropdown, then click their leaderboard row to read individual conversations.
- **Custom reporting period** — Use the date range picker to select an arbitrary window (e.g., a promotional campaign period) and switch granularity to **Week** or **Month** for a high-level view.
