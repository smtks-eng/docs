Las **Integraciones** conectan tu agente de IA a plataformas y servicios externos. Así es como vinculas tu agente a aplicaciones de mensajería, herramientas de soporte al cliente, CRMs y otros sistemas que tu equipo ya usa.

## Tipos de integración disponibles

### Canales de mensajería
- **WhatsApp (Meta)** — Despliega tu agente en WhatsApp Business vía la API de Meta
- **Freshchat** — Integra con Freshchat para flujos de soporte en vivo
- **Zendesk** — Conecta con Zendesk para ticketing y soporte en vivo

### Canales web
- **Web Bubble** — Embebe un widget de chat en tu sitio web
- **External Chat URL** — Comparte un enlace de chat independiente

<Tip>
Para instrucciones paso a paso de configuración por canal, consulta la sección **Guías de Integración** al final del menú lateral.
</Tip>

## Configurando una integración

<Steps>
  <Step title="Selecciona una integración">
    Haz clic en la integración que deseas configurar. Verás las credenciales y opciones de configuración requeridas.
  </Step>
  <Step title="Ingresa tus credenciales">
    Proporciona las API keys, tokens o detalles de autenticación requeridos por la plataforma. Estos se cifran y almacenan de forma segura.
  </Step>
  <Step title="Configura los canales">
    Una vez conectada la integración, configura los canales específicos (ej. un número de WhatsApp, una bandeja de Freshchat) que deseas que use tu agente.
  </Step>
  <Step title="Asigna a tu agente">
    En la sección de **Canales** del tab Conectar, asigna el canal configurado a tu agente.
  </Step>
  <Step title="Prueba la conexión">
    Usa el **Playground** o envía un mensaje de prueba a través del canal para verificar que la integración funciona correctamente.
  </Step>
</Steps>

## Notas clave

- Las integraciones son a **nivel de organización** — múltiples agentes pueden usar la misma integración conectada.
- Las **credenciales se cifran** — las API keys y tokens se almacenan de forma segura.
- Algunas skills (como la **Escalación Humana**) solo aparecen cuando hay una integración compatible (Freshchat o Zendesk) conectada.
- Desconectar una integración afectará a **todos los agentes** que la usan.
