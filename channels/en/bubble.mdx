The **Web Bubble** is a floating chat widget you can embed directly on your website. It gives visitors instant access to your AI agent from any page—no redirects, no separate apps.

## Key Features

- **Embedded experience:** The bubble lives on your site, opening a chat window inline.
- **Customizable appearance:** Configure brand colors, launcher style, and welcome messages from the Console.
- **File attachments:** Users can send images and documents directly in the chat.
- **Escalation support:** When a conversation is escalated to a human agent, users see a visual pill indicating the handoff.

### Message Grouping

Consecutive messages from the same sender are **visually grouped** together. This means:
- Only the **last message** in a group shows a timestamp, keeping the view clean.
- Bubble shapes adapt: the first and last messages in a group have slightly different border-radius for a modern, threaded appearance.
- A small gap separates different sender groups, while messages within the same group are closely spaced.

### Human Agent Identity

When a conversation is escalated and a human agent responds through the bubble, end users see:
- The **agent's name** displayed above the first message in a group.
- An **avatar with the agent's initials** next to the last message in the group (e.g., "JD" for Jane Doe).

This helps end users know exactly who they are speaking with after an escalation, creating a more transparent and personal support experience.

<Tip>
  Human agent names are automatically pulled from the team member's profile in Semantiks. Make sure your team members have their names configured under **Settings**.
</Tip>

## How to Set Up

<Steps>
  <Step title="Open Bubble channel">
    In **AI Agent > Connect > Channels**, open the Bubble card.
  </Step>
  <Step title="Generate credentials/snippets">
    The Console provides the snippet and keys needed for embedding.
  </Step>
  <Step title="Add snippet to your website">
    Paste the script where your implementation team manages site code.
  </Step>
  <Step title="Customize appearance">
    Configure brand settings like launcher style and welcome behavior.
  </Step>
</Steps>

## Key Notes

- The bubble works on any website that supports JavaScript embeds.
- Conversations started through the bubble appear in the **Conversations** panel for supervision.
- The bubble supports the same escalation, follow-up, and memory features as other channels.
