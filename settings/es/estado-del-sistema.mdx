# Estado del Sistema

La página de **Estado del Sistema** te ofrece una vista rápida del rendimiento de cada servicio principal de la plataforma Semantiks. Puedes acceder desde **Configuración → Estado del Sistema**.

## Verificaciones de Salud

La página muestra el estado actual de los siguientes servicios:

| Servicio | Descripción |
| --- | --- |
| **Servicio de Mensajería de Integraciones** | Gestiona el envío de mensajes a través de tus canales conectados (WhatsApp, Freshchat, Zendesk, etc.). |
| **API de Consola** | Alimenta la consola de Semantiks — la aplicación web que estás usando en este momento. |
| **Servicio de Agente** | Ejecuta el razonamiento y la generación de respuestas de tu agente de IA. |

Cada servicio muestra:

- **Estado** — si el servicio está operativo o presenta problemas.
- **Verificado el** — la fecha y hora en que se verificó el estado por última vez.

<Tip>
  Si algún servicio aparece con problemas, espera unos minutos y actualiza la página. Si el problema persiste, contacta a soporte.
</Tip>

<Warning>
  Si ves un error de "Error al obtener el estado de salud", verifica tu conexión a internet e inténtalo de nuevo.
</Warning>
