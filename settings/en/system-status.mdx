# System Status

The **System Status** page gives you a quick overview of how each core service in the Semantiks platform is performing. You can access it from **Settings → System Status**.

## Health Checks

The page displays the current status of the following services:

| Service | Description |
| --- | --- |
| **Integrations Messaging Service** | Handles message delivery across your connected channels (WhatsApp, Freshchat, Zendesk, etc.). |
| **Console API** | Powers the Semantiks console — the web application you are using right now. |
| **Agent Service** | Runs your AI agent's reasoning and response generation. |

Each service shows:

- **Status** — whether the service is operational or experiencing issues.
- **Checked At** — the date and time the status was last verified.

<Tip>
  If any service appears unhealthy, wait a few minutes and refresh the page. If the issue persists, contact support.
</Tip>

<Warning>
  If you see a "Failed to fetch health status" error, check your internet connection and try again.
</Warning>
